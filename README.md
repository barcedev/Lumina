# <img src="https://cdn.modrinth.com/data/cached_images/acc2d4fcfa257ae08296c6c6e47c561b5a931b54_0.webp" width="24" /> Lumina
A niche, lightweight, fullbright gamma utility mod.
Allows for better visibility in the dark, as well as configurable brightness levels for the Nether and End dimensions. 

💡 **Press `G` to toggle fullbright!**

## ✅ Features
- **Sodium Integration**
  - Config is available in Video Settings
- **Dimension Brightness**
  - Customize your brightness and fullbright values for each dimension
  - Automatically adjusts when traveling between dimensions
- **Actionbar Notifications**
  - Minimal and Simple Actionbar Message styles
  - Toggleable

## 🚧 Planned Features
- Customizable "Minimal" and "Simple" Action bar messages
- Smooth Transition

I don't want to add too much bloat to this mod so features will be limited!
